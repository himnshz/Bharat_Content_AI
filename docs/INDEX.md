## 🆘 Troubleshooting

### Common Issues
1. **Backend won't start** - Check Python version, virtual environment
2. **Frontend won't start** - Check Node version, reinstall dependencies
3. **AI generation fails** - Verify API keys in `.env`
4. **Database errors** - Check DATABASE_URL in `.env`
5. **Redis connection fails** - Start Redis server

### Getting Help
- Check relevant guide in this index
- Review API documentation
- Check error logs in terminal
- Open issue on GitHub
